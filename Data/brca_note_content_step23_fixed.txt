## Step 2.1: Filter & Impute Methylation

```
# Remove probes with >20% missing values
missing_pct <- rowSums(is.na(meth_mat)) / ncol(meth_mat)
meth_keep <- meth_mat[missing_pct < 0.2, ]

# Impute remaining missing values with KNN
library(impute)
meth_imputed <- impute.knn(as.matrix(meth_keep), k = 10)$data

cat("Methylation probes before:", nrow(meth_mat), "after filter+impute:", nrow(meth_imputed), "\n")
```

**🧠 What you're doing:** Removing bad methylation probes (too many missing values) and filling in the remaining gaps using similar probes' values (KNN imputation). Think of KNN as: "find the 10 most similar probes and average their values to fill the gap."

**🎯 Why:** MOFA2 can't handle missing values. Methylation data often has ~5% missing values. We remove probes that are mostly missing, then intelligently fill the rest rather than throwing away whole patients.

---

## Step 2.2: Select Top Variable Features Per Omics

```
# RNA: keep top 8000 most variable genes
gene_vars <- apply(rna_mat, 1, var, na.rm = TRUE)
rna_top8k <- rna_mat[names(sort(gene_vars, decreasing = TRUE)[1:8000]), ]

# Methylation: keep top 8000 most variable probes
meth_vars <- apply(meth_imputed, 1, var, na.rm = TRUE)
meth_top8k <- meth_imputed[names(sort(meth_vars, decreasing = TRUE)[1:8000]), ]

# CNV: keep top 3000 most variable genes (CNV is less dense)
cnv_vars <- apply(cnv_mat, 1, var, na.rm = TRUE)
cnv_top3k <- cnv_mat[names(sort(cnv_vars, decreasing = TRUE)[1:3000]), ]

# RPPA: keep all (~200 proteins) — already low-dimensional
cat("RNA:", nrow(rna_top8k), "Methyl:", nrow(meth_top8k), "CNV:", nrow(cnv_top3k), "RPPA:", nrow(rppa_mat), "\n")
```

**🧠 What you're doing:** Picking the most informative features from each data layer. Genes/probes that barely change across patients carry little information. RPPA has only ~200 proteins, so we keep them all. RNA and methylation are reduced to 8000, CNV to 3000.

**🎯 Why:** MOFA2 training time scales with the number of features. 8000 genes + 8000 probes + 3000 CNV + 200 RPPA = ~19,200 features, which is manageable. Keeping everything (e.g., 20,000 genes) would be slow and noisy.

## Step 2.3: Align All Patients and Build MAE

```
deduplicate_by_patient <- function(mat) {
    patient_ids <- extract_patient_id(colnames(mat))
    dup_idx <- duplicated(patient_ids)
    if (sum(dup_idx) > 0) cat("Removing", sum(dup_idx), "duplicate samples from", deparse(substitute(mat)), "\n")
    mat[, !dup_idx, drop = FALSE]
}

# Recalculate common patients (safe for fresh session)
common_patients <- Reduce(intersect, lapply(
    list(rna_top8k, meth_top8k, cnv_top3k, rppa_mat),
    function(m) unique(extract_patient_id(colnames(m)))))
cat("Common patients:", length(common_patients), "
")

rna_final <- deduplicate_by_patient(rna_top8k)
meth_final <- deduplicate_by_patient(meth_top8k)
cnv_final <- deduplicate_by_patient(cnv_top3k)
rppa_final <- deduplicate_by_patient(rppa_mat)

# Subset to common patients and align column order
for (nm in c("rna_final", "meth_final", "cnv_final", "rppa_final")) {
    mat <- get(nm)
    pids <- extract_patient_id(colnames(mat))
    assign(nm, mat[, pids %in% common_patients])
}

common_ordered <- sort(common_patients)
align_by_patient <- function(mat, ref_order) {
    pids <- extract_patient_id(colnames(mat))
    mat[, order(match(pids, ref_order))]
}

rna_final  <- align_by_patient(rna_final, common_ordered)
meth_final <- align_by_patient(meth_final, common_ordered)
cnv_final  <- align_by_patient(cnv_final, common_ordered)
rppa_final <- align_by_patient(rppa_final, common_ordered)

# Remove duplicate feature names
rna_final <- rna_final[!duplicated(rownames(rna_final)), ]
meth_final <- meth_final[!duplicated(rownames(meth_final)), ]
cnv_final <- cnv_final[!duplicated(rownames(cnv_final)), ]
rppa_final <- rppa_final[!duplicated(rownames(rppa_final)), ]

# Assign patient IDs as column names
colnames(rna_final) <- extract_patient_id(colnames(rna_final))
colnames(meth_final) <- extract_patient_id(colnames(meth_final))
colnames(cnv_final) <- extract_patient_id(colnames(cnv_final))
colnames(rppa_final) <- extract_patient_id(colnames(rppa_final))

stopifnot(colnames(rna_final) == common_ordered)
stopifnot(colnames(meth_final) == common_ordered)
stopifnot(colnames(cnv_final) == common_ordered)
stopifnot(colnames(rppa_final) == common_ordered)

# Build MultiAssayExperiment
assay_list <- list(
    RNA = as.matrix(rna_final),
    Methyl = as.matrix(meth_final),
    CNV = as.matrix(cnv_final),
    RPPA = as.matrix(rppa_final)
)

clinical_mat <- as.data.frame(clinical)
clinical_ids <- extract_patient_id(clinical_mat[[1]])
dup_idx <- duplicated(clinical_ids)
clinical_mat <- clinical_mat[!dup_idx, , drop = FALSE]
clinical_ids <- clinical_ids[!dup_idx]
rownames(clinical_mat) <- clinical_ids
clinical_mat <- clinical_mat[common_ordered, , drop = FALSE]

mae <- MultiAssayExperiment(
    experiments = ExperimentList(assay_list),
    colData = DataFrame(clinical_mat)
)

saveRDS(mae, "Data/brca_mae.rds")
summary(mae)
cat("Final patient count:", ncol(mae), "\n")
```

**🧠 What you're doing:** Building a single organized R object (MultiAssayExperiment) that contains all 4 omics layers + clinical data, with every column matching the same patient. It's like a binder with 4 tabs, each tab sorted the same way.

**🎯 Why:** The MAE is the standard format MOFA2 expects. By building it once and saving to `Data/brca_mae.rds`, you can reload it in any later session without repeating the alignment.

## Step 3.2: Train MOFA Model

```
Sys.setenv(RETICULATE_PYTHON = "C:/Users/hp/AppData/Local/Programs/Python/Python312/python.exe")

library(MOFA2)

mae <- readRDS("Data/brca_mae.rds")

# Build named list of matrices from MAE
assay_names <- names(experiments(mae))
cat("Experiments:", assay_names, "\n")

mofa_data <- list()
for (nm in assay_names) {
    mat <- as.matrix(assay(experiments(mae)[[nm]]))
    dup <- duplicated(rownames(mat))

cat("Data views:", names(mofa_data), "\n")
cat("Features:", sapply(mofa_data, nrow), "\n")
cat("Samples:", unique(sapply(mofa_data, ncol)), "\n")

data_opts <- get_default_data_options(mofa_obj)
model_opts <- get_default_model_options(mofa_obj)
model_opts$num_factors <- 15
model_opts$spikeslab_factors <- FALSE
model_opts$ard_factors <- TRUE

train_opts <- get_default_training_options(mofa_obj)
train_opts$convergence_mode <- "fast"
train_opts$maxiter <- 1000
train_opts$seed <- 42

mofa_obj <- prepare_mofa(mofa_obj, data_options = data_opts,
                          model_options = model_opts,
                          training_options = train_opts)

dir.create("models", showWarnings = FALSE)
mofa_trained <- run_mofa(mofa_obj, outfile = "models/brca_mofa2.hdf5")
saveRDS(mofa_trained, "models/brca_mofa_trained.rds")
cat("MOFA training complete. Model saved.\n")
```

**🧠 What you're doing:** Training the MOFA model — it's like running a "pattern detector" across all 4 omics simultaneously. MOFA finds 15 latent "factors," each representing a coordinated pattern (e.g., "these 200 genes go up, those 50 methylation sites go down, and this protein drops"). Each patient gets a score for each factor.

**🎯 Why:** This is the core of the analysis. Instead of testing 8000 genes one at a time, MOFA compresses the data into 15 meaningful signals. Each factor might represent a biological process (immune response, cell cycle, hormone signaling) that we can then test for survival association.

**⚠️ Expect this to take 15-30 minutes with ~700 patients and 4 omics layers.**

## Step 5.1: Cox Regression — Overall Survival (OS)

```
factor_scores <- get_factors(mofa_trained)[[1]]
colnames(factor_scores) <- paste0("Factor", 1:ncol(factor_scores))

# Overall survival endpoints
os_time <- as.numeric(ifelse(
    clinical_mat$vital_status == "DECEASED",
    clinical_mat$days_to_death,
    clinical_mat$days_to_last_followup
))
os_event <- ifelse(clinical_mat$vital_status == "DECEASED", 1, 0)

cat("OS — Deaths:", sum(os_event), "Censored:", sum(!os_event), "\n")

cox_os <- data.frame()
for (f in colnames(factor_scores)) {
    fmla <- as.formula(paste0("Surv(os_time, os_event) ~ `", f, "`"))
    cox <- coxph(fmla, data = data.frame(os_time, os_event, factor_scores))
    s <- summary(cox)
    cox_os <- rbind(cox_os, data.frame(
        Factor = f, Endpoint = "OS",
        HR = exp(coef(cox)), p_value = coef(s)[, "Pr(>|z|)"],
        CI_lower = exp(confint(cox)[1]), CI_upper = exp(confint(cox)[2])
    ))
}
cox_os$p_adj <- p.adjust(cox_os$p_value, method = "BH")
write.csv(cox_os, "logs/cox_os.csv", row.names = FALSE)

cat("\n=== COX RESULTS — OVERALL SURVIVAL ===\n")
for (i in 1:nrow(cox_os)) {
    cat(sprintf("%-10s HR=%5.3f [%5.3f-%5.3f]  p=%6.4f  adj=%6.4f\n",
        cox_os$Factor[i], cox_os$HR[i], cox_os$CI_lower[i], cox_os$CI_upper[i],
        cox_os$p_value[i], cox_os$p_adj[i]))
}
```

**🧠 What you're doing:** Testing each of the 15 MOFA factors for association with how long patients live (overall survival). This is identical to the LUSC analysis — we look for factors where patients with high scores die sooner (HR > 1) or later (HR < 1).

**🎯 Why:** Overall survival is the gold standard endpoint. With ~1100 patients (vs 370 in LUSC), we have more statistical power. More factors may reach significance. This is the first hurdle: if no factors predict OS, the deeper analyses in Part 2 become critical.

## Step 5.2: Cox Regression — Recurrence-Free Survival (RFS)

```
# Recurrence-free survival: time to recurrence or death, whichever comes first
# TCGA may have days_to_recurrence or days_to_tumor_progression
rfs_cols <- grep("recurrence|progression|recur|progres",
                 names(clinical_mat), value = TRUE, ignore.case = TRUE)
cat("RFS-related columns:", paste(rfs_cols, collapse = ", "), "\n")

# Check which columns exist
for (col in rfs_cols) {
    vals <- clinical_mat[[col]]
    cat(col, "- non-NA:", sum(!is.na(vals)), "\n")
}

# Best guess for RFS columns
recur_time_col <- intersect(rfs_cols,
    grep("days_to_recurrence|days_to_tumor_progression|recurrence_days|progression_days",
         rfs_cols, value = TRUE, ignore.case = TRUE))

recur_event_col <- intersect(rfs_cols,
    grep("recurrence_status|progression_status|recurred|progressed",
         rfs_cols, value = TRUE, ignore.case = TRUE))

if (length(recur_time_col) > 0 && length(recur_event_col) > 0) {
    rfs_time <- as.numeric(clinical_mat[[recur_time_col[1]]])
    rfs_event <- as.numeric(clinical_mat[[recur_event_col[1]]])

    # For patients without recurrence, use OS as censored time
    na_rfs <- which(is.na(rfs_time))
    rfs_time[na_rfs] <- os_time[na_rfs]
    rfs_event[na_rfs] <- os_event[na_rfs]

    cat("RFS — Events:", sum(rfs_event, na.rm = TRUE), "Censored:", sum(rfs_event == 0, na.rm = TRUE), "\n")

    cox_rfs <- data.frame()
    for (f in colnames(factor_scores)) {
        fmla <- as.formula(paste0("Surv(rfs_time, rfs_event) ~ `", f, "`"))
        cox <- coxph(fmla, data = data.frame(rfs_time, rfs_event, factor_scores))
        s <- summary(cox)
        cox_rfs <- rbind(cox_rfs, data.frame(
            Factor = f, Endpoint = "RFS",
            HR = exp(coef(cox)), p_value = coef(s)[, "Pr(>|z|)"],
            CI_lower = exp(confint(cox)[1]), CI_upper = exp(confint(cox)[2])
        ))
    }
    cox_rfs$p_adj <- p.adjust(cox_rfs$p_value, method = "BH")
    write.csv(cox_rfs, "logs/cox_rfs.csv", row.names = FALSE)

    cat("\n=== COX RESULTS — RECURRENCE-FREE SURVIVAL ===\n")
    for (i in 1:nrow(cox_rfs)) {
        cat(sprintf("%-10s HR=%5.3f [%5.3f-%5.3f]  p=%6.4f  adj=%6.4f\n",
            cox_rfs$Factor[i], cox_rfs$HR[i], cox_rfs$CI_lower[i], cox_rfs$CI_upper[i],
            cox_rfs$p_value[i], cox_rfs$p_adj[i]))
    }
} else {
    cat("RFS columns not found. Available progression columns:\n")
    print(rfs_cols)
    cat("If RFS data unavailable, use OS only for all downstream analyses.\n")
}
```

**🧠 What you're doing:** Testing the same 15 factors for recurrence-free survival — time until the cancer comes back or the patient dies. RFS is often more sensitive than OS because it captures earlier events.

## Step 6.1: Enhanced Continuous Cox with LASSO + Clinical Adjustment + Multiple Splits

### Why this step?

Part 1 split patients into "High" vs "Low" with median cut — wasting information. Continuous scores preserve full signal. LASSO pre-filters to reduce multiple testing. Clinical covariates (age, stage) remove known survival effects so subtle factor signals can emerge. Extreme tertile splits maximize contrast.

```
Sys.setenv(RETICULATE_PYTHON = "C:/Users/hp/AppData/Local/Programs/Python/Python312/python.exe")
setwd("C:/Users/hp/Dev/playground/PO3")
library(stringr)
library(MultiAssayExperiment)
library(MOFA2)
library(survival)
library(glmnet)
library(ggplot2)
library(survminer)

mae <- readRDS("Data/brca_mae.rds")
mofa_trained <- readRDS("models/brca_mofa_trained.rds")
clinical_mat <- as.data.frame(colData(mae))

factor_scores <- get_factors(mofa_trained)[[1]]
colnames(factor_scores) <- paste0("Factor", 1:ncol(factor_scores))

os_time <- as.numeric(ifelse(clinical_mat$vital_status=="DECEASED", clinical_mat$days_to_death, clinical_mat$days_to_last_followup))
os_event <- ifelse(clinical_mat$vital_status=="DECEASED", 1, 0)

# Clinical covariates
age <- as.numeric(clinical_mat$age_at_diagnosis)
stage_char <- clinical_mat$pathologic_stage
stage <- ifelse(grepl("III|IV", stage_char), "Late", "Early")

cat("OS — Deaths:", sum(os_event), "Censored:", sum(!os_event), "\n")

                      family="cox", alpha=1, nfolds=10)

    selected_factors <- names(selected_idx)
    cat(" (min):", paste(selected_factors, collapse=", "), "\n")
}

# B: Continuous Cox (all 15)
cat("\n=== CONTINUOUS COX (OS) ===\n")
results_os <- data.frame()
for (f in paste0("Factor", 1:15)) {
    fmla <- as.formula(paste0("Surv(os_time, os_event) ~ `", f, "`"))
    cox <- coxph(fmla, data=data.frame(os_time, os_event, scale(factor_scores)))

    results_os <- rbind(results_os, data.frame(Factor=f, HR=exp(coef(cox)),
        p_value=coef(s)[,"Pr(>|z|)"], CI_lower=exp(confint(cox)[1]), CI_upper=exp(confint(cox)[2])))
}
results_os$p_adj <- p.adjust(results_os$p_value, method="BH")
for (i in 1:nrow(results_os))
    cat(sprintf("%-10s HR=%6.3f [%5.3f-%5.3f]  p=%6.4f  adj=%6.4f\n",
        results_os$Factor[i], results_os$HR[i], results_os$CI_lower[i],
        results_os$CI_upper[i], results_os$p_value[i], results_os$p_adj[i]))
write.csv(results_os, "logs/cox_continuous_os.csv", row.names=FALSE)

# C: Cox + clinical covariates
cat("\n=== COX ADJUSTED FOR AGE + STAGE ===\n")
clin_data <- data.frame(os_time, os_event, age=scale(age),
    stage=factor(stage), scale(factor_scores))
clin_data <- clin_data[complete.cases(clin_data), ]
cat("With clinical data:", nrow(clin_data), "\n")

results_adj <- data.frame()
for (f in paste0("Factor", 1:15)) {
    fmla <- as.formula(paste0("Surv(os_time, os_event) ~ `", f, "` + age + stage"))
    cox <- tryCatch(coxph(fmla, data=clin_data), error=function(e) NULL)
    if (is.null(cox)) next
    s <- summary(cox)
    results_adj <- rbind(results_adj, data.frame(Factor=f, HR=exp(coef(cox))[1],
        p_value=coef(s)[1,"Pr(>|z|)"], CI_lower=exp(confint(cox))[f,1], CI_upper=exp(confint(cox))[f,2]))
}
results_adj$p_adj <- p.adjust(results_adj$p_value, method="BH")
for (i in 1:nrow(results_adj))
    cat(sprintf("%-10s HR=%6.3f [%5.3f-%5.3f]  p=%6.4f  adj=%6.4f\n",
        results_adj$Factor[i], results_adj$HR[i], results_adj$CI_lower[i],
        results_adj$CI_upper[i], results_adj$p_value[i], results_adj$p_adj[i]))
write.csv(results_adj, "logs/cox_adjusted_clinical_os.csv", row.names=FALSE)

# D: KM curves (median + extreme tertile for top 3)
top3 <- results_os$Factor[order(results_os$p_value)][1:3]
for (f in top3) {
    med <- median(factor_scores[,f], na.rm=TRUE)
    group <- ifelse(factor_scores[,f] > med, "High", "Low")
    fit <- survfit(Surv(os_time, os_event) ~ group)
    p <- ggsurvplot(fit, pval=TRUE, risk.table=TRUE,
                    title=paste(f, "- median split (p=", round(results_os$p_value[results_os$Factor==f], 4), ")"))
    png(paste0("figures/km_median_", f, ".png"), width=8, height=6, units="in", res=300); print(p); dev.off()
}
cat("Saved 3 median-split KM curves\n")

# Extreme tertiles
for (f in top3) {
    q <- quantile(factor_scores[,f], c(1/3, 2/3), na.rm=TRUE)
    group <- rep("Mid", length(factor_scores[,f]))
    group[factor_scores[,f] <= q[1]] <- "Low"
    group[factor_scores[,f] > q[2]] <- "High"
    group <- factor(group, levels=c("Low","Mid","High"))
    idx <- group != "Mid"
    fit <- survfit(Surv(os_time[idx], os_event[idx]) ~ droplevels(group[idx]))
    p <- ggsurvplot(fit, pval=TRUE, risk.table=TRUE, title=paste(f, "- extreme tertiles"))
    png(paste0("figures/km_tertile_", f, ".png"), width=8, height=6, units="in", res=300); print(p); dev.off()
}
cat("Saved 3 extreme-tertile KM curves\n")
```

**🧠 What you're doing:** Four complementary approaches in one script: (A) LASSO picks the most promising factors, (B) continuous Cox uses full numeric scores, (C) adjusts for age and stage to remove noise, and (D/E) visualize the best factors with two split strategies.

**🎯 Why:** LASSO reduces the 15 factors to a handful (fewer tests = more power). Clinical adjustment accounts for the fact that older patients and late-stage patients die faster regardless of biology. Extreme tertiles remove the middle 33% where signal is weakest — making the High vs Low contrast sharper.

## Step 6.2: Stratify by Molecular Subtype

### Why this step?

BRCA has 5 clinically distinct subtypes (LumA, LumB, HER2-E, Basal, Normal-like). They have different treatments, different prognoses, and different biology. ## Step 6.3: Random Survival Forests

### Why this step?

Cox assumes a straight line: double the factor score = double the risk. Real biology has thresholds, plateaus, and interactions. RSF automatically handles these without you specifying them.

```

mofa_trained <- readRDS("models/brca_mofa_trained.rds")

os_event <- ifelse(clinical_mat$vital_status=="DECEASED", 1, 0)

rf_data <- data.frame(scale(factor_scores), time=os_time, status=os_event)

    scale_fill_manual(values=c("TRUE"="steelblue","FALSE"="grey70")) +
    labs(title="RSF Variable Importance", x="", y="VIMP")
ggsave("figures/rsf_vimp.png", p, width=7, height=5)
```

**🎯 Why:** RSF is a robustness check. If the same factors that were borderline significant in Cox are also the top 3 in RSF, that's converging evidence. If RSF picks completely different factors, there may be non-linear effects Cox missed.

## Step 6.4: Add Somatic Mutation Data

## Step 6.5: Nomogram with Bootstrap Validation

```

}
```

**🧠 What you're doing:** Building a clinical prediction chart. Add up points for age, stage, and the LASSO-selected factors. The total maps to a 1-year and 3-year survival probability. Bootstrap validates the C-index (discrimination) and calibration (accuracy).

**🎯 Why:** A nomogram is the most publication-ready output. If the C-index > 0.70 and the calibration curve follows the 45° line, this is a clinically useful model that could be presented to doctors. The slope shrinkage tells you if you're overfitting (shrinkage < 0.8 = too complex).
